# Докеризация приложения КупиПодариДай

Проект содержит докеризированный сервис "КупиПодариДай", состоящий из:
- Бэкенд-сервиса на Node.js
- Базы данных PostgreSQL
- Фронтенда на React (раздаётся через nginx)

## Структура проекта

- `backend/` - исходный код бэкенд-сервиса
- `frontend/` - исходный код фронтенд-приложения
- `docker-compose.yml` - конфигурация для запуска всех сервисов

## Запуск проекта

1. Скопируйте `env.example` в `.env` и заполните значения переменных окружения
2. Запустите проект: `docker-compose up -d`
3. Фронтенд будет доступен на http://localhost:8081
4. Бэкенд будет доступен на http://localhost:4000
